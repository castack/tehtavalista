# Tehtävälista


Tehtävälistasovellus jolla voidaan hallita tehtäviä ja luokitella niitä kiirellisyyden perusteella. 

**Ominaisuudet:**
  - Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
  - Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan tehtäviä. 
  - Käyttäjä pystyy luokittelemaan tehtäviä kiirellisyyden perusteella. 
  - Käyttäjä pystyy jakamaan tehtäviä muiden käyttäjien kesken. 
  - Käyttäjä pystyy määrittämään tehtävän valmiuden.
  - Käyttäjä pystyy hakemaan tehtäviä hakusanalla. 
  - Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja sekä tietoja eri käyttäjien lisäämistä tehtävistä.

Pääasiallinen tietokohde on käyttäjäkohtainen tehtävä. Toissijainen tietokohde on tehtävän kiirrellisyys ja tehtävän valmius.
